# i3-kali-config
Contains my i3wm configuration used for Kali

### File Placement

**i3status.conf:**
> /etc/i3status.conf

**config:**
> /etc/i3/config


### Applications Required

- terminator
    - Terminal Emulator
- mate
    - mate-panel
    - mate-volume-control-applet
    - mate-power-manager
    - mate-screenshot
